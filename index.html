Fundraising Automation · Multi-million pound charitable enterprise · Nov 2025
Build: 2h Saves: 6d/yr

Grant Writing Automation

The fundraiser faced a recurring burden: identifying suitable grant funders, then completing lengthy applications that repeatedly requested identical organisational information. Each application consumed a full working day - researching eligibility criteria across multiple funder databases, adapting project descriptions to different question formats, and re-entering standard details. I built a system that stores the organisation's core information and uses examples from previous applications to maintain consistent tone. When seeking funding, the fundraiser describes project requirements and the system searches for matching opportunities, then generates draft applications adapted to each funder's specific questions whilst preserving the organisation's established voice. Applications that previously required eight hours now need ten minutes of review and submission. With six applications annually, this recovers six working days - time now spent building funder relationships rather than processing paperwork.

Production Tracking Automation · Publishing house · Nov 2025
Build: 4h Saves: 13d/yr

Production Tracking System

The editor couldn't answer basic questions about book status. Information lived scattered across five spreadsheets - editorial progress in one, proofreading stages in another, indexing elsewhere. Finding out whether a title would meet its print deadline required searching multiple files and asking several people. I built a unified tracker showing every book's stage, responsible person, and deadline in one place. A traffic light system flags titles at risk, and percentage completion compares against target progress. Staff update it manually as work completes, avoiding the complexity of automated integrations whilst maintaining current visibility. The editor previously spent two hours weekly chasing status information across disconnected systems. That time is eliminated. The system didn't prevent measurable failures - books weren't missing print slots - but it transformed operational chaos into clarity, allowing the editor to focus on editorial decisions rather than detective work.

Sales Data Restructuring · Multi-million pound charitable enterprise · Dec 2025
Build: 20h Saves: 13d/yr

Sales Data Automation

Management needed comprehensible dashboards to inform stock and pricing decisions. The underlying sales data made this impossible. The spreadsheet contained 43,000 formulas sprawled across multiple sheets, with duplicated data and calculations buried in beige and green cells. Most of the data went ignored because interpretation required forensic effort. I restructured the entire database, rotating the layout from horizontal to vertical and eliminating the formula layers. All calculations now happen in Looker Studio rather than the spreadsheet. Instead of copying and pasting through complex sheets, staff use clickable filters to target specific data instantly. The finance team previously spent two hours weekly navigating the labyrinth. That's eliminated. More importantly, non-technical staff can now extract meaningful insights within minutes. The data hasn't changed, but accessibility has - transforming an unusable archive into a working decision-making tool.
